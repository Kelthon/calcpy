# calcPy

## About

calcPy is a simple calculator created in python

## How to Install

### On Windows

``` bash
    git clone <repository name>
```

#### Config virtual enviorement

``` bash
    python3 -m venv venv
    ./venv/Scripts/activate
    pip install PySimpleGui
```
